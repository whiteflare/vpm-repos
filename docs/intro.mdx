## VCCへの追加方法

[VCCに追加する](https://whiteflare.github.io/vpm-repos/addrepos.html) をクリックすることで追加できます。<br />
Repository JSON は次の場所にあります。次のURLをVCCのSettingsにて追加することでもリポジトリを追加できます。
```
https://whiteflare.github.io/vpm-repos/vpm.json
```
